# Brand Assets

Place the following logo files in this folder. The app references them by these exact filenames:

- `logo-primary.svg` — P-mark + "PRESSURE-IT" + "PREMIUM PROPERTY CARE" (for dark backgrounds)
- `logo-onecolour.svg` — White/one-colour version
- `icon-mark.svg` — The P-mark alone (used for favicon, PWA icon, social avatar)
- `logo.png` — Raster fallback

If a file is missing, the app falls back to a text-based logo.
